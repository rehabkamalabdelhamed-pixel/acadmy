# Acadmy Learning Platform - كل ما تحتاج إلى معرفته 📚

## نظرة عامة
منصة تعليمية متكاملة تدعم 3 أدوار: **مشرف** 👨‍💼 | **مدرسون** 👨‍🏫 | **طلاب** 🎓

---

## 🚀 البدء السريع

### 1. تشغيل الخادم (Backend)
```bash
cd backend
npm install
npm start
```
السيرفر يعمل على `http://localhost:3000`

### 2. فتح الواجهة الأمامية
افتح أي متصفح وانتقل إلى:
- **تسجيل الدخول**: `acadmy_project/login/login.html`
- **التسجيل الجديد**: `acadmy_project/register/register.html`

## 👥 الحسابات الجاهزة

### المشرف (Admin)
- **البريد**: `admin12345@gmail.com`
- **كلمة المرور**: `Admin@123`
- **الصلاحيات**: إدارة المدرسين والطلاب والفيديوهات

### المدرسون (Teachers)
| البريد الإلكتروني | كلمة المرور |
|---|---|
| `warda12345@gmail.com` | `Teacher@123` |
| `ahlam12345@gmail.com` | `Teacher@123` |
| `nourhan12345@gmail.com` | `Teacher@123` |
- **الصلاحيات**: إضافة وحذف فيديوهات، مشاهدة إحصائيات طلابهم

### الطلاب (Students)
- **التسجيل**: أي بريد إلكتروني جديد + كلمة مرور
- **الصلاحيات**: مشاهدة الفيديوهات ومتابعة التقدم

## 🎯 لوحات التحكم

### 📊 لوحة المشرف (Admin Dashboard)
`/admin/dashboard.html`
- **الإحصائيات**: عرض إجمالي المستخدمين والمدرسين والطلاب والفيديوهات
- **إدارة المدرسين**: قائمة بجميع المدرسين
- **إدارة الطلاب**: البحث والتحكم في الطلاب
- **إدارة الفيديوهات**: حذف أي فيديو من المنصة

### 🎓 لوحة المدرس (Teacher Dashboard)
`/teacher/home.html`
- **الفيديوهات الخاصة**: عرض فيديوهاتك فقط
- **إضافة فيديو جديد**: 
  - عنوان، وصف، رابط YouTube
  - اختيار التصنيف (برمجة، JavaScript، ذكاء اصطناعي)
- **البحث والحذف**: إدارة كاملة للفيديوهات
- **الإحصائيات**: عدد الفيديوهات المضافة

### 👨‍🎓 لوحة الطالب (Student Dashboard)
`/student/dashboard.html`
- **مشاهدة الفيديوهات**: جميع الفيديوهات المتاحة
- **متابعة التقدم**: 
  - نسبة المشاهدة لكل فيديو
  - إجمالي التقدم الشامل
- **الفلترة والبحث**: 
  - البحث بالعنوان
  - تصفية حسب التصنيف
- **الإحصائيات الشخصية**:
  - عدد الفيديوهات المشاهدة
  - عدد الفيديوهات المكتملة (80% فأكثر)

## 🔌 API Endpoints

### 🔐 المصادقة (Authentication)
| الطريقة | المسار | الوصف |
|--------|--------|--------|
| POST | `/login` | تسجيل الدخول |
| POST | `/register` | تسجيل حساب جديد |

### 📹 الفيديوهات (Videos)
| الطريقة | المسار | الوصف |
|--------|--------|--------|
| GET | `/videos` | الحصول على جميع الفيديوهات |
| GET | `/videos?teacher=email` | فيديوهات مدرس معين |
| GET | `/videos?category=برمجة` | فيديوهات حسب التصنيف |
| GET | `/videos/:id` | فيديو محدد |
| POST | `/videos` | إضافة فيديو جديد (مدرسون فقط) |
| DELETE | `/videos/:id` | حذف فيديو |

### 📊 متابعة التقدم (Progress)
| الطريقة | المسار | الوصف |
|--------|--------|--------|
| GET | `/progress/:studentEmail` | تقدم طالب معين |
| POST | `/progress` | تسجيل متابعة فيديو |

### 📈 الإحصائيات (Stats)
| الطريقة | المسار | الوصف |
|--------|--------|--------|
| GET | `/admin/stats` | إحصائيات شاملة |
| GET | `/admin/users` | قائمة المستخدمين |

## 💾 تخزين البيانات

البيانات تُحفظ في ملفات JSON:
- **`users.json`**: المستخدمين وكلمات المرور
- **`videos.json`**: الفيديوهات والمعلومات
- **`progress.json`**: متابعة تقدم الطلاب

> **ملاحظة**: في الإنتاج، استخدم قاعدة بيانات مثل MongoDB أو MySQL

## 📝 الأمثلة على الاستخدام

### إضافة فيديو (كمدرس)
```javascript
fetch('http://localhost:3000/videos', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    title: 'مقدمة في JavaScript',
    description: 'شرح أساسيات...',
    url: 'https://youtube.com/...',
    category: 'JavaScript',
    teacher: 'warda12345@gmail.com'
  })
})
```

### تسجيل متابعة فيديو (كطالب)
```javascript
fetch('http://localhost:3000/progress', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    student: 'student@email.com',
    videoId: '1234567890',
    status: 'completed',
    progress: 100
  })
})
```

## ❓ المشاكل الشائعة

### الخادم لا يعمل؟
```bash
# تأكد من npm install
npm install

# جرب على منفذ آخر (افتح server.js وغير PORT)
PORT=5000 npm start
```

### الفيديوهات لا تظهر؟
- تأكد من أن السيرفر يعمل (`http://localhost:3000/ping`)
- تحقق من CORS في السيرفر
- افتح console في المتصفح لرؤية الأخطاء

### localStorage لا تعمل؟
- فقط localhost و HTTPS تدعمها
- تأكد من عدم استخدام الوضع الخاص (Private Mode)
